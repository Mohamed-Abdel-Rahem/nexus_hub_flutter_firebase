Key Features 🚀
Authentication: Secure Login/Signup flow using Firebase Auth.

Real-time Database: Cloud Firestore for dynamic data management.

State Management:  Optimized UI updates with structured state logic).

API Integration: Weather services integrated using Dio & Singleton Pattern.

Error Handling: Custom exception handling layer for seamless user experience.

Tech Stack 🛠️
Frontend: Flutter SDK (Dart)

Backend: Firebase (Auth, Firestore, Storage)
Design Pattern: Singleton, Repository Pattern.
